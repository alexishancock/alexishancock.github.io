Fun project highlight: Internet in a Box

I unwrapped my new Raspberry Pi 3 and immediately put it to work. I installed and flashed an image on an SD Card from the Internet in a Box project. I was interested in how to configure an at home set up for people who don't have access to a proper internet connection. To do things like research, study various subjects, and when connected online, download more information.

I used:

  • Raspberry Pi 3
  • CanaKit Raspberry Pi 3 with 2.5A Micro USB Power Supply
  • iUniker Raspberry Pi 3 B+ Case
  • 64 GB MicroSD card. (I recommend 32 GB since I ran into issues with this size with FAT32 formatting)
